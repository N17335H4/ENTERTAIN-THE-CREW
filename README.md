# ENTERTAIN-THE-CREW

Using Minimax algorithm to build an unbeatable Tic Tac Toe game powered by AI, MARS COLONIZATION PROGRAM.

Demo: https://karunakaushik.github.io/game.io

Using React-App

Steps to run the web app on terminal:

      2. NPM start
